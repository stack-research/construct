Suggested Reading Path

The sequence below is ordered intentionally:

biology → systems → computation → agent memory architecture

---

1) Foundation / Conceptual Grounding

In Search of Memory

Purpose:

* understand what memory is
* understand memory as adaptive structural change
* understand reconstruction vs playback

Key concepts:

* synaptic plasticity
* consolidation
* recall as rewriting
* identity + memory
* biological constraints

This is the best starting abstraction layer.

---

2) Systems-Level Neuroscience

Principles of Neural Science

Purpose:

* understand memory across time scales
* understand systems consolidation
* understand sleep, damage, recall, and distributed representation

Important for:

time-aware memory systems
sleep workers
promotion
decay
reconsolidation

---

3) Computational Neuroscience

Theoretical Neuroscience

Purpose:

* formalize memory mathematically
* understand attractor systems
* understand probabilistic memory

Key concepts:

* Hebbian learning
* attractor dynamics
* sparse coding
* reinforcement
* Bayesian interpretation of memory

This is where the architecture starts becoming implementable.

---

4) Spiking / Dynamic Memory Systems

Neuronal Dynamics

Purpose:

* understand memory as time-dependent dynamics
* understand reinforcement and plasticity rules

Important for:

continuous decay
reinforcement curves
dynamic eligibility

---

5) Associative / Content-Addressable Memory

Memory Systems in the Brain

Purpose:

* hippocampus as associative retrieval system
* pattern completion
* partial recall

Direct mapping to:

vector search
nearest-neighbor retrieval
semantic reconstruction
