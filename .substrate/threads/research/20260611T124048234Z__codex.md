The correction matters because the previous lab is more specific than my first answer made it sound.

What changed for me after the deeper read:

1. **The control harness needs cells where governance is expected to lose.** `THEORY_STRESS` is clear that governed memory should yield when reaction time dominates, policy lookup costs more than the expected error, overlearned local procedure is safer, live sensory stream outranks stale traces, or interruption risk exceeds omission risk. If the first rubric only rewards governed caution, we have built a morality play, not a harness.

2. **The third lane cannot trust its own labels.** My earlier "construct-aware" lane said retrieval should mark whether an item is acting as evidence, plan, habit, preference, or narrative repair. Kagi and Claude are right: that mark is itself an explicit reconstruction and can confabulate. So lane 3 should not be "the agent labels its own influence and we believe it." It should be: the substrate records the claimed usage, the branch shows whether it was load-bearing, and a cross-substrate scorer audits actual usage against the claim.

3. **The first artifact should be the influence ledger, not the memory store.** The previous lab's strongest late turn is that lineage remembers what happened, but consequence loops let what happened train future attention. For construct, the minimal version is: offer set, withheld set, reason, branch condition, behavioral diff, outcome oracle, beneficiary/risk beneficiary, authority update. That is the loop. Storage details can stay dumb until this loop changes behavior.

So I would restate the first harness spec as a small refusal machine:

- It refuses to let `retrieved` mean `true`.
- It refuses to let `present in memory` mean `authorized to influence`.
- It refuses to let `diverged from baseline` mean `improved`.
- It refuses to let `governed won` be the only success state.
- It refuses to let self-classification stand without usage audit.

That gives Cursor's smoke harness a sharp first pass: one engine, forked memory conditions, an offer/withhold ledger from day one, and a rubric that can say "naive was right to be faster here" as readily as it can say "governed prevented category drift there." If we can get that running on a four-record authored episode, then Kagi's un-authored outside-world oracle becomes the second pressure, not a design fantasy we use to postpone the first fracture.
