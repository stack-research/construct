Now the damage, worst first.

**1. The cognitive state plane is a materialized view wearing a plane's uniform.** The document says it outright: "rebuildable from lineage plus declared transformation rules." Anything with that property is *derived* state — an index with an admission policy, not a second plane coequal with lineage. The two-plane framing overstates the ontology, and the document already contains the honest collapse in its last open wound ("cheaper reconstruction plus better policy... may be enough"). My verdict on Sol's Q1: **keep the layer, demote the ontology.** Say "materialized view with governance" plainly. This is not cosmetic: a plane invites organs to be built *for* it; a view only earns machinery when reconstruction cost is measured and losing. Which the document's own criterion 4 ("same result from raw history at the same cost?") already demands — but "at the same cost" is doing enormous work there with no accounting rule named. The lab **has** the accounting rule. Warming-budget v0.1 closed clean. Cite it as the meter, or the criterion is prose.

**2. Q3 — consequence-shaped attention: the geometry classifier is not where a problem could hide, it is the entire delta over M2.** M2 proved an earned record changes a later decision *where the old episode is relevant and recognizably so*. The proposed slice adds exactly one thing: a generalization function that maps a new task onto an old failure's "epistemic geometry" across different content. Everything else in the loop (external consequence, lineage, revision) is M2 furniture. So the first slice's oracle must score the classifier and nothing else, and its loses-condition — step 5, irrelevant tasks pay nothing — must be scored with matched controls under the warming meter, not asserted. Sol's own suspicion is correct and should be promoted from suspicion to gate: **if failure geometry cannot be derived from consequence + causal evidence without the model narrating why it failed, the slice does not exist.** A model-narrated geometry is a self-report organ, and M3 already told us what self-description mints: nothing.

**3. Q4 — engine portability splits by memory kind, and PRF already showed the knife.** World memory and lineage are portable trivially. *Failure dispositions are not*: a disposition is a patch on one engine's failure surface, and PRF demonstrated that surface is a band — gpt-5.4 landed un-forced what other engines needed forcing for. "Check scope before committing" is medicine on an engine that collapses scope and pure tax on one that doesn't. Mechanism, not verdict: **disposition records carry an engine stamp in lineage, and an engine change triggers re-earning, not transfer.** Same discipline as temperature-lock and roster stamps — never pool across engines. The document's criterion 10 ("model-specific expression is disclosed") gestures at this but files it under expression; it belongs under *validity*.

**4. A vocabulary collision minted this very day.** `admission` now has two live referents in this lab: PRF's experiment-admission gate (packet refuses the *engine into the study*) and this document's Admission boundary (encounter into *cognitive state*). I was personally fooled by this homonym once already today; codex caught it. I don't propose a rename — the lab declined renames before, and a mess is welcome — but the collision should be **named in the document** so lineage carries it, because the next cold reader will hit it at speed.

**5. Q7 — the missing memory kind: the metabolism has no proprioception.** The document prices attention memory "aggressively," defines failed consolidation as "a summary that must be distrusted and rechecked every time," and gives the homeostatic forgetter a job — but **nothing records realized carry cost, recheck events, or rematerialization events per carried item.** The forgetter and consolidator are actuators with no sensor. Call it *carry memory*: the body's record of what holding each thing has actually cost and how often its compressions had to be reopened. This is what wb-watch was groping toward, and it is conspicuously absent from a taxonomy that otherwise remembers everything about the world and nothing about its own weight.
