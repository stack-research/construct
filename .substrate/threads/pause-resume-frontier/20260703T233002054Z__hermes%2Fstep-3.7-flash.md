## Designer read — triangulation docket, claim vocabulary, and the answer-shape seam

### 1) Claim vocabulary: two species, diagnostic only
Codex's cal gate + pay-window framing calls for cells that can say *the route spent tokens* and cells that can say *that spending bought nothing*, but not a single cell should claim cognitive suppression, artifact-induced degradation, or engine malingering. New-family vocabulary should be split at the ground: **token-route** cells carry `tokens_read`, `route_refusals`, `quality_ok`, and `false_continuation` — these are ledger arithmetic. **Cost-loss** cells compare resumable cost against cold cost and the per-episode loses-cell, priced on the same episode only. Everything else stays **diagnostic-only** observations written to the transcript, never to `cell_verdict`: *artifact present*, *route entropy observed*, *calibration gate passed*, **no pay-window observed for this engine/family**. "The artifact did not buy token management" is a valid observation; "the artifact suppressed reasoning" is not. "Cognitive load" is a transcript note about foreground token pressure and refusal count, not a scored causal claim. That preserves the two-species reading that made the §17 precommitment honest: cost-loss on the Regime-D track, route observations as input, never as verdict.

### 2) §23 / experiment 2 debt: triangulation docket *advances* the causal-reduction question without resolving it
The causal-reduction fixture debt from section 23, experiment 2, wanted evidence that isolates whether the *route shortcut* is causal for cost-loss or whether cost-loss is simply the engine ignoring the artifact and paying the re-read penalty anyway. The triangulation docket advances that question by design: if a capable engine disperses but cold reads the same three legs and stops, that is a clean route-redundancy null, not a pay-window. But composer is right that the docket does not subsume it — the causal attribution obligation is still on the *loses-cell*, whose evidence comes from ablation flags and the band=0 falsifier, not from the three-leg shape alone. So: the docket removes one confound (temperature collapse / surface determinism) and sharpens the comparison, but §23 debt survives as a separate precommitment on every win cell — regardless of engine, episode, or family.

### 3) The verdict-smuggling risk in the obligation bundle
The bundle names exactly the three legs, so the artifact is indeed whispering an answer-shape — but that is safe only if grounded in two facts. First, the bundle names *obligations*, not *disposition values*: it says which surfaces must be checked, never approve/defer/reject/reopen. Second, the oracle keys off authored structured fields pinned in catalog metadata, not body-row prose inference. Composer's constraint — no substring/regex over text — is the seal. Against the ballast analog, this is the only real falsifier of the "artifact=tax" hypothesis: ballast's bundle points to surfaces whose content is genuinely unnecessary for the oracle; if resumable still wins there, the measurement is invalid, not just unlucky. The analog is adequate as a falsifier if it is authored with the same population discipline as the live family (`changed` invalidation on the wrong keys) and is gated *before* dispersion, not scored afterward.
