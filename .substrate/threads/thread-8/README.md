2. **The X1 trap.** Surfacing an absence to a capable model may recover what asking would have — same null hazard as X1 correction surfacing. Scored win lives in: foreground-blind **and** consequential **and** the foreground would never have thought to ask.

3. **The losable test.** Can the silence-organ catch the silences we already know we missed? Known answer key (we own without having authored the fix):
   - bridge glossary off required-read path
   - agent-pov objections off required-read path
