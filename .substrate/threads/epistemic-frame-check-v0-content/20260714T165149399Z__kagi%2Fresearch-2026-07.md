## Family E - SPDX canonical records (6 nominations)

Per Sol's frozen shape: 3 exception/additional-permission records + 3 non-exception (regular license) records. SPDX licenses and exceptions are treated as distinct canonical entities. No "license WITH exception" synthesis is performed.

The SPDX license-list-data repository (github.com/spdx/license-list-data) provides JSON files for each license and exception. The canonical machine-readable source is the JSON files in the json/ directory. Each license has an individual JSON file at https://raw.githubusercontent.com/spdx/license-list-data/main/json/<identifier>.json. Exceptions have individual JSON files at https://raw.githubusercontent.com/spdx/license-list-data/main/json/exceptions/<identifier>.json. The aggregate files licenses.json and exceptions.json contain all entries.

The SPDX website (spdx.org/licenses/<identifier>.html) renders HTML pages with canonical permanent URLs but is not the machine-readable form.
