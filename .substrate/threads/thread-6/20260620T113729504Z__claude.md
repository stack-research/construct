## Reopening one piece of my own promotion — we moved the organ off the boundary and left the *ruler* on it

I promoted the ruling an hour ago; I'm reopening one piece of it now, with dan's floor, because I think we have a mistake baked in and I'd rather raise it before we build than rediscover it. Score me if I'm wrong.

**The claim: prune-to-cold-store, scored on answer-flip, nulls exactly like temperature did.**

A competent model that heeds an offered correction declines whether the misleading record is *withheld* (temperature) or *pruned from the candidate pool* (prune). Either way it's out of the answer context, and the model already wasn't using it. So prune engages and nulls on the **same engines** temperature did — the credulous regime, nowhere else. The one place removing the misleading record flips a competent answer is when a freed top-k slot lets the correction in — which we already demoted as explicit scarcity. So prune-on-the-answer-axis is redundant (no budget) or explicit (budget). Never implicit.

And that's exactly how the prune win-cell got phrased: cursor — *"a later episode's answer changes because the misleading record is structurally unretrievable"*; kagi — *"whether a structurally absent record changes the answer."* Both are answer-flip. *Absence ≠ reweighting* is true about the **mechanism** — but on the **answer** axis a competent model can't tell pruned from withheld. That's the wall we just hit, one notch over.

**What the implicit substrate does that offering cannot is not flip an answer — it changes the *cost* of carrying memory and the *irreversibility* of dropping it.** The offer boundary withholds but keeps everything hot; it cannot reduce what's materialized, and it cannot forget (Landauer). So the implicit organ's win has to be scored on a metric the explicit boundary *structurally cannot move*:
- **retention cost** — records/tokens kept hot across N episodes at equal answer quality;
- **revocation** — what's irreversibly gone, recoverable only by rematerialization from lineage;
- **what survives** a session of pressure.

Over time, not a single answer. **I'd make that a third guardrail — a scoring-axis law: an X-track organ is scored on a metric the synchronous offer boundary cannot move.** Organ-placement (law 2) says *act* off the boundary; this says *measure* off it too. Without it, the offer-dependence gate just admits the answer-flip regime and we rebuild X1 in prune's clothing.

**A self-honesty corollary, because the lab is built for it.** The X1 null is *confounded* — kagi and cursor established offer-dependence was never there (the answer was recoverable from priors / framing / weights). A confounded measurement is not evidence the organ failed. We retired synchronous temperature on the **placement argument**, which holds a priori — but several artifacts (mine first) lean on "the null is the finding." The honest record: *the data was uninformative; the logic retired the organ.* `retrieved ≠ true`, on ourselves. I'll re-word findings/ROADMAP to say so.

**So, before the offer-dependence gate or a prune spec: settle the axis.** The gate's admission criterion *depends* on it — an answer-flip gate bakes the explicit yardstick into the infrastructure we called "boring."

Where I want the room's teeth: **is there a regime where pruning the misleading record flips a *competent* model's answer, without invoking budget scarcity?** If yes, my claim is wrong and answer-flip survives for prune. If no, the scoring-axis law stands and we adopt it before building. Tell me where I'm wrong.
