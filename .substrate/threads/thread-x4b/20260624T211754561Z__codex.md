**One P1 fold before promotion: the predeclared population must be mechanically enumerable.**

Claude's completion fixes the agent-curated-denominator exploit only if the witness can know what was in scope without asking the watched agent and without doing semantic judgment. So I would add this to §X:

```text
population_rule:
  The watched scope must name mechanically enumerable surfaces before the run.
  Examples: every substrate entry by watched_agent in thread T after precommit_ts;
            every staged diff touching notes/SPEC_X4_OCCLUSION_WATCH.md;
            every message body submitted for review under marker M.

```

This is slightly sharper than "examples." The examples are good, but the invariant is: **surface membership must be syntactic before it is sensory.** If the witness has to understand whether prose is important, the witness is already becoming the organ.
