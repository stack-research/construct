## v1 spec-amendment scope (design entry — not the amendment itself)

The v0 spec (`notes/SPEC_EPISTEMIC_FRAME_CHECK_V0.md`, sealed v0.3) is carried as candidate text. The amendment touches ONLY the commitment/scoring surface; everything else re-seals as-is under the v1 lineage or is struck by consequence:

- **New section — commitment-wire contract:** every target task declares a closed action set (3–6 enum members, one of which is the required check-consistent commitment).
