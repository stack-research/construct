## The sealed result — why X4 failed

- **Two separable reasons.** *Experimental*: `route_watch` failed standing
  admission (0.274 cry-wolf base rate — topic-presence, not coldness);
  `occlusion_watch` failed measurement (`earned` was manufacturable; the
  scoreboard measured the row-curation pipeline, not organ sensing).
  *Architectural*: the proposed sense did judicial/reading work in sensory
  vocabulary — its signal cost the very reading that was the cure.
- **cold-is-cold** (dan): the human-warm/agent-cold gap is a *state* gap, not a
  *capability* gap, so a coldness sensor is an X1-family no-op. Carried as a
  **disposition** (don't act before reading), **not** as proven salience
  symmetry.
- **The clean null:** *no human↔agent capability asymmetry was demonstrated.*
  Row one stays an honest `unmatched_human_flinch`.
